# ADA M5 · Sorting Lab

## Estructura del repositorio

Este repositorio contiene las soluciones a los problemas del módulo 5. Cada sección tiene su propia carpeta con un `README.md` que incluye el análisis y código correspondiente.

| Carpeta | Contenido |
|---|---|
| [`probabilistico/`](./probabilistico/README.md) | Análisis probabilístico y algoritmos aleatorios |
| [`awful_sorting/`](./awful_sorting/README.md) | Algoritmos de ordenamiento con mal desempeño |
| [`fun_sort/`](./fun_sort/README.md) | Algoritmos de ordenamiento creativos |
